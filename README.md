## Made mostly with "themes.vscode.one"

# Installation:
1. Open the extensions sidebar on Visual Studio Code.
2. Search for Dark Star Pro Theme.
3. Click Install.
                                    (OR)
1. Press F1.
2. Type "ext install vijval.dark-star-pro".
3. Smash the enter button.
